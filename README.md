<h1>Rodar Localmente</h1>
<p>Para executar o pojeto localmente, siga-os os passos a baixo:</p>
<h3>Clone o Projeto</h3>
<pre>git clone https://github.com/LucasStorck/Yogalax.git</pre>

<h3>Istale as Depedências</h3>
<pre>npm install</pre>

<h3>Execute</h3>
<pre>npm run dev</pre>
